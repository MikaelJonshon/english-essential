# Assets do Dashboard do Professor

Diretorio reservado para recursos estaticos da area administrativa.

## Estrutura atual

- `images/`: imagens futuras da interface administrativa.
- `audio/`: audios futuros da interface administrativa.

## Estado atual

Ainda nao ha assets de producao versionados nestas pastas.

Elas existem para manter o mesmo padrao de organizacao usado no dashboard do aluno e para facilitar expansoes futuras, como:

- icones ou ilustracoes do painel;
- imagens de comunicados;
- audios de notificacao administrativa;
- materiais de suporte para professores.
