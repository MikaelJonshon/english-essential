# Audios da Licao Principal 1

Audios usados na Licao Principal 1 do Modulo 1.

## Uso

Esse lote inclui palavras, frases curtas e falas-base para:

- listening;
- speaking;
- repeticao de pronuncia;
- associacao entre audio, texto e imagem.

## Padrao atual

- formato: `.mp3`;
- nomes em ingles;
- palavras separadas por `_`;
- arquivos leves para carregamento sob demanda.

Exemplos:

- `boy.mp3`
- `hello.mp3`
- `the_boy_is_reading.mp3`
- `they_are_running.mp3`
- `she_is_cooking.mp3`

## Observacao

Ao cadastrar exercicios no Supabase, use caminhos relativos previsiveis para facilitar manutencao, por exemplo:

```text
dashboard-aluno/assets/m1/l1/audio/the_boy_is_reading.mp3
```
