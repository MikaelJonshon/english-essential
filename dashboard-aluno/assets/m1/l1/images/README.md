# Imagens da Licao Principal 1

Imagens usadas na Licao Principal 1 do Modulo 1.

## Uso

Esses arquivos apoiam exercicios de:

- pronuncia;
- vocabulario;
- leitura;
- escrita;
- associacao entre imagem, palavra e frase.

## Padrao atual

- formato: `.jpg`;
- nomes em ingles;
- palavras separadas por `_`;
- uma imagem por conceito, pessoa, acao ou frase visual.

Exemplos:

- `boy.jpg`
- `girl_reading.jpg`
- `man_drinking.jpg`
- `they_are_running.jpg`
- `woman_writing.jpg`

## Observacao

Ao cadastrar exercicios no Supabase, use caminhos relativos previsiveis para facilitar manutencao, por exemplo:

```text
dashboard-aluno/assets/m1/l1/images/boy_reading.jpg
```
